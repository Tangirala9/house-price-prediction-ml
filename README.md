# House Price Prediction using Machine Learning

## Project Overview

This machine learning project predicts house prices based on:

- Area (Square Feet)
- Number of Bedrooms

The model is built using Linear Regression.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn

---

## Dataset

| Area | Bedrooms | Price |
|--------|--------|--------|
| 800 | 1 | 2000000 |
| 1000 | 2 | 3000000 |
| 1500 | 3 | 4500000 |

---

## Machine Learning Workflow

1. Load Dataset
2. Feature Selection
3. Train-Test Split
4. Linear Regression Model
5. Model Training
6. Prediction
7. Evaluation using Mean Squared Error

---

## Features

- Area
- Bedrooms

## Target

- House Price

---

## Sample Prediction

Input:

Area = 1800

Bedrooms = 3

Output:

Predicted House Price ≈ 5500000
